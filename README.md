# NexusPHP

PT站源码主程序，nexusphp.v1.5.beta5.20130110更新（小樱修改版）

-----
## 服务器配置推荐：

- MySQL 版本：5.4 （5.1.66及以上）
- 系统：Linux / Windows 
- PHP 版本：v5.3.8（v5.3.3以上，v7.0 以下）

请确保PHP版本为5.3.3之上，否则可能导致种子无速度，无法上传种子，等等兼容性问题。
memcached服务器组件+memcache的PHP支持
